⚡ One-time setup
1
Enable Google APIs — In Google Cloud Console, enable Google Slides API and Google Drive API.
2
Create OAuth Client ID — APIs & Services → Credentials → OAuth 2.0 Client ID → Web application. Add your hosting domain to Authorised JavaScript Origins.
3
Prepare your template — Import your PPTX into Google Slides and replace data values with {{PLACEHOLDER}} tags (see guide above). Copy the file ID from the URL.
4
Deploy — Drop index.html, api/brain.js, and vercel.json into Vercel or Netlify (free).
