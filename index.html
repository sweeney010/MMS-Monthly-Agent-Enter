⚡ One-time setup
1
Enable Google APIs — In Google Cloud Console, enable Google Slides API, Google Drive API and Google Sheets API.
2
Create OAuth Client ID — APIs & Services → Credentials → OAuth 2.0 Client ID → Web application. Add your Vercel domain to Authorised JavaScript Origins.
3
Prepare your template — Replace data values with {{PLACEHOLDER}} tags (see guide above). Copy the Slides file ID from the URL.
4
Deploy — Drop index.html, api/brain.js, and vercel.json into Vercel (free).
